Manager Agent Responsibilities

Version Control & Publish

  2. Stage only transcripts/index.html, the new lecture .md, and the new lecture .html.

  4. Report back with post-push git status, commit hash, changed files, and verification notes (links checked, calendar entry added, missing list updated).
