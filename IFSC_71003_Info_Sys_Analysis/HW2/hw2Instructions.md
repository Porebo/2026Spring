## Appendix: Quantitative Evaluation Methodology

*Reference: Jan 27, 2026 04:24 PM Lecture Recording - "Website Evaluation Quality Metrics"*

In the context of the course, **Quantitative Evaluation** means judging a system's quality based on measurable data (numerical scores and percentages) across specific "Dimensions."

### Dimensions and Metrics Table

| Dimension | Definition | Quantitative Metric (Calculation) |
| :--- | :--- | :--- |
| **Usefulness** | Benefit provided to the user. | (Sessions resulting in success/links) ÷ (Total sessions) × 100 |
| **Usability** | Ease of navigation/interaction. | Ratio of instructional success to help requests received. |
| **Findability** | Search engine visibility. | % of time site appears in Top 10 results for a set of queries. |
| **Accessibility** | Support for users with disabilities. | % of images with `alt` text or % compliance with scanners. |
| **Credibility** | Authoritativeness/Believability. | % of pages showing site structure (e.g., breadcrumbs). |
| **Value** | Advantage provided to the public. | % of faculty learning materials available without passwords. |
