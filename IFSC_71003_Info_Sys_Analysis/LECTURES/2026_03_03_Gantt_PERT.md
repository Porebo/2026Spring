## Gantt Chart Overview
- History: origin with Karol Adamiecki (“harmonograms”) in 1896; Henry Gantt popularized them circa 1910. Cameras/industrial revolution context noted.
- Example: Hoover Dam/Mike O’Callaghan–Pat Tillman Memorial Bridge uses a bronze Gantt chart plaque (rare physical artifact).
- Chart anatomy: bars indicate tasks with start/end dates; arrows can show dependencies; color coding distinguishes phases.
- Mentioned onlineGantt.com (free tool producing .gant files) but for HW7 students will hand-draw their charts.

## PERT Diagram Overview
- Acronym history: Program Evaluation Research Task ? Program Evaluation Review Technique.
- Developed for U.S. Navy Polaris (1957); later used for Olympic planning.
- Activity-on-node graphs capture dependencies and durations, enabling critical path identification.
- Walkthrough example: activities A–H with durations; forward scan (earliest start) and backward scan (latest start) computed via video tutorial; critical path identified as A?C?E?G?H with 20-week minimum schedule.
