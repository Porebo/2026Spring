## Grading: The 50–100 Scale

### The Problem with 0–100

Daniel demonstrated with two hypothetical students to show why 0–100 grading is unfair:

- **Student M:** 100s on everything in Course 1, 0s on everything in Course 2
  - Grades: A + F → GPA = **2.0** (C)
- **Student N:** 100s on everything in both courses for first half, 0s for second half
  - Each course averages to 50% → F + F → GPA = **0.0**
- Both learned the same amount, but radically different GPAs!

### The 50–100 Fix

- **Student M:** 100s in Course 1, 50s in Course 2 → A + F → GPA = **2.0**
- **Student N:** Average of 100s and 50s = 75% in each course → C + C → GPA = **2.0**
- Now both students get the same GPA — **fair!**

Reference: "The Case Against the Zero" article (linked in syllabus)
