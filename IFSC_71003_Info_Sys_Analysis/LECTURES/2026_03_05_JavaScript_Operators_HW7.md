- **Arithmetic & assignment review:** `+`, `-`, `*`, `/`, `%`, `**`, along with compound assignments like `+=` update variables in place.

- **Increment behavior:** pre-increment (`++x`) changes the value before evaluation; post-increment (`x++`) returns the original value first.
- **Logical truth tables:** `&&`, `||`, and `!` were mapped against boolean inputs to show when expressions short-circuit.
- **Type insight tools:** `typeof` reveals the runtime type, while `parseInt()`/`parseFloat()` convert strings to numeric forms instead of relying on implicit coercion.
