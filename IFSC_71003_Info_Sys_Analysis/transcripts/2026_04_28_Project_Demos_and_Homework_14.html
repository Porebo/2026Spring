2. AI Tools for Website Development

AI-supported building workflow
  • Students were encouraged to use AI tools such as ChatGPT, Gemini, and DeepSeek to generate and refine website content.
  • AI can accelerate page drafting, layout generation, and text production for project sites.
  • Some tasks still require manual intervention, especially page-to-page navigation and linking adjustments.
AI helps with development speed, but it does not automatically handle every structural detail of a production-ready website.

3. Infinity Free Account Warning

Hosting reliability reminder
  • Infinity Free accounts may be recycled when inactive for long periods.
  • Students should log in regularly to keep accounts active.
  • A link from the student homepage to the project site should be present and working.
  • Students were advised to maintain backups in case accounts are suspended during heavy development.
Protect your work with frequent backups. Free hosting interruptions can happen unexpectedly near deadlines.

4. Homework 14 Answer Page Requirements

Submission expectations
  • The Homework 14 answer page should explain what was done to develop or finalize the project website.
  • Students must include a direct link to the live project site.
  • The answer should document concrete progress, not only a placeholder statement.
Minimum expectation: a short implementation summary plus a working project URL.
