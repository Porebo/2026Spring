4. Gantt Chart Deep Dive

15:30–38:30
  • History: Karol Adamiecki (harmonograms, 1896) vs. Henry Gantt (1910). Industrial-age context and European origins.
  • Example: Hoover Dam / Mike O’Callaghan–Pat Tillman Bridge uses a bronze Gantt chart plaque (rare physical artifact).
  • Structure: bars represent tasks with start/end dates; arrows show dependencies; color coding helps categorize work.
  • Tool highlight: onlinegantt.com (free, saves .gant files). For HW7, students will hand-draw instead.

5. PERT Diagram Primer

38:30–55:30
  • PERT = Program Evaluation Review Technique (formerly “Research Task”); created for the U.S. Navy Polaris project (1957) and later used for Olympic planning.
  • Activity-on-node graph showing durations and dependencies; supports the critical path method.
  • Class example: activities A–H. Conducted forward scan (earliest start) and backward scan (latest start) using a tutorial video.
  • Critical path identified as A ? C ? E ? G ? H with 20-week minimum completion time.
