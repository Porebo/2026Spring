Task 1: Form Creation
Rideshare Driver Log — A practical shift-logging form built with 8 distinct HTML components:
  1. <input type="date"> — Shift date picker
  2. <input type="time"> — Start time selector
  3. <select> — Platform dropdown (Uber / Lyft / Both / Other)
  4. <input type="text"> — Area driven (free text)
  5. <input type="number"> — Ride count with min/max
  6. <input type="radio"> — Shift type (Morning / Afternoon / Evening / Late Night)
  7. <input type="checkbox"> — Extras (cash tips, long ride, surge pricing)
  8. <textarea> — Free-form notes
Submissions POST to a custom PHP handler that sanitizes input, appends entries to a server-side JSON file, and confirms the save. A separate PHP results page retrieves and displays all logged shifts in a sortable table with aggregate stats.
