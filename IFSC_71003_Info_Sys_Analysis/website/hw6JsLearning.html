const perDay = (raw / 7).toFixed(1);
readingEstimate.textContent = `Estimated pacing: ${perDay} pages/day...`;
readingGoal.addEventListener("input", updateReadingEstimate);
AI Learning Transcript (Brief Socratic Style)
Student: What does toFixed(1) do in simple terms?
AI: It keeps one digit after the decimal so the number is easier to read.
Student: Why divide by 7?
AI: To convert a weekly total into an estimated daily pace.
Student: What is textContent changing here?
AI: It updates the visible text in the estimate area of the page.
Student: What does addEventListener("input", ...) do?
AI: It reruns the function every time the user changes the input value.
Student: Why is this useful for users?
AI: They get immediate feedback without clicking submit or refresh.
