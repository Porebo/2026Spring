Task 6: Reflection

← Back to Homework 6

For this homework, I learned that AI helps most when I use it as an iteration partner instead of a one-shot answer engine. I got better results by requesting small, testable changes and validating each one quickly.

I also learned that prompt precision matters more than prompt length. Asking brief, targeted questions about one line of JavaScript produced clearer learning than asking broad questions about the whole script.

A practical benefit was speed: AI reduced setup time for repetitive structure, while I focused on decisions, verification, and content quality. The downside is that AI can produce overbuilt responses unless I constrain scope and ask for concise output.

Overall, the most valuable workflow was: plan task outputs, generate minimal code, test behavior, then refine. That process helped me understand JavaScript behavior and still keep ownership of the final result.
