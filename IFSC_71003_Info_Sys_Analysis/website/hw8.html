Part 2: Improvement Recommendations
  • Use one digital triage form so students are routed correctly the first time.
  • Enable advising notes visibility for registrar and financial aid to reduce repeat checks.
  • Create appointment batching for related services (advising + registrar in same slot).
  • Add campus wayfinding map with shortest routes between high-traffic offices.
Part 3: JavaScript Functions and Loops Recap
function totalDistance(segments) {
  let total = 0;
  for (let i = 0; i < segments.length; i++) {
    total += segments[i];
  }
  return total;
}

const walkingSegments = [450, 380, 500, 380, 620];
const campusDistance = totalDistance(walkingSegments); // 2330 ft
