Q1: Three Manual (Non-AI) Changes to Sandbox

← Back to Homework 5 Open Updated Sandbox
What I Changed by Hand
  1. Added stronger validation on Full Name using maxlength and pattern restrictions.
  2. Added a new radio-button input group for Preferred Study Mode.
  3. Added a live character counter for the Notes textarea with a 300-character limit.
Manual Change Snippets

Change 1: name validation constraints

<input type="text" id="fullName" name="fullName"
  maxlength="60" pattern="[A-Za-z ]{2,60}"
  title="Use letters and spaces only (2-60 characters)" required>

Change 2: added radio-button section

<label>Preferred Study Mode</label>
<input type="radio" name="studyMode" value="Solo" required> Solo
<input type="radio" name="studyMode" value="Group"> Group
<input type="radio" name="studyMode" value="Mix of both"> Mix of both

Change 3: notes character counter

function updateCount() {
  notesCount.textContent = `${notes.value.length} / 300 characters`;
}
notes.addEventListener("input", updateCount);
