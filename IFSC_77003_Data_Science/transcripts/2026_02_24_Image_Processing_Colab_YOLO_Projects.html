IFSC 77003 · Data Science & Technologies · Week 6

Image Processing Practice, YOLO Detection, and Project Planning

February 24, 2026 Dr. Serhan Dagtas · TA Sharifat · Dr. Elizabeth Pierce UALR
Google Colab Lab Core Operations Filtering and Edges Color Block Detection YOLO Object Detection Project Planning

This class was run as a practical lab session around the Blackboard Google Colab notebook, with live parameter tuning and examples spanning channel handling, intensity transforms, filtering, frequency analysis, and pretrained object detection.

Notebook-Centered Workflow

Students were asked to open the same Colab notebook and sample files while the team demonstrated each step. The objective was hands-on familiarity rather than abstract theory.

Course Intent

Use image processing to prepare raw visual data, then feed improved representations into downstream machine learning or deep learning tasks.

Foundational Image Operations

Color channels and grayscale

  • Loaded images with OpenCV and visualized RGB and grayscale variants.
  • Discussed how channel intensity differences influence object characteristics.

Point transforms and contrast controls

  • Negative transform and brightness/contrast tuning.
  • Alpha/beta style adjustment and clipping behavior.
  • Gamma-like nonlinear transformation.
  • Histogram plotting and equalization for contrast redistribution.

Filtering, Edge Detection, and Frequency Domain

The lab compared mean and Gaussian smoothing with multiple kernel sizes, then moved to sharpening and Laplacian behavior. It showed the practical trade-off between denoising and detail retention.

TechniqueTypical EffectRisk if Overused
Mean/Gaussian smoothingNoise reduction and visual smoothingEdge/detail loss
Sharpening/LaplacianEdge emphasis and structural visibilityAmplified artifacts/noise
Low-pass filtering (frequency)Suppress high-frequency detailOver-blurred structures
High-pass filtering (frequency)Highlight boundaries/fine detailsNoisy outputs

The same operation can produce very different results depending on the image content and parameter settings.

Live Lab Observation

Mini Project: Color Block Detection

A lightweight OpenCV project demonstrated HSV-style thresholding for red/green/blue segmentation, followed by mask construction, contour/bounding-box annotation, and color labeling. This gave a clean example of translating pixel thresholds into object-level results.

Pretrained Object Detection with YOLO

The session then applied a pretrained YOLO model (Ultralytics workflow) to detect people and objects in sample images, review confidence values, and visualize result boxes.

Threshold Tuning Insight

Higher confidence threshold yields fewer, more certain detections; lower threshold yields more detections, including weaker or potentially false positives.

This directly connected image preprocessing quality with model-level detection outcomes and real deployment tuning decisions.

Assignments and Project Direction

  • Optional/bonus Unit 6 tasks were highlighted (basic processing, retrieval, cut detection, and coding exercises).
  • Students were invited to submit bonus work directly to the instructor.
  • Thursday session focus: selecting data science project topics.
  • Kaggle was recommended for curated image datasets and challenge ideas.
Actionable Next Step

Run the notebook independently before the next class, vary parameters, and document what improves or degrades detection quality for your chosen image type.
